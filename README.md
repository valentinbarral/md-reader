# MD Reader

A modern, feature-rich Markdown reader application with a beautiful and customizable interface.

## Demo

Try it online: [https://valentinbarral.github.io/md-reader/](https://valentinbarral.github.io/md-reader/)

## Features

- **Drag & Drop**: Simply drag and drop your Markdown files to start reading
- **Multiple Themes**: Choose from Light, Dark, Sepia, Gray, and Contrast themes
- **Customizable Fonts**: Select from various font families optimized for reading
- **Adjustable Settings**: Customize font size, content width, and view mode
- **Table of Contents**: Automatic outline generation for easy navigation
- **Multi-language Support**: Available in English, Spanish, and Galician
- **Pagination Mode**: Switch between continuous and paged viewing modes
- **Syntax Highlighting**: Code blocks are beautifully highlighted

## Usage

1. Open `index.html` in your web browser
2. Drag and drop a Markdown file (`.md` or `.markdown`) or click "Select file"
3. Customize your reading experience using the settings panel
4. Navigate through the document using the table of contents sidebar

## Technologies

- Vanilla JavaScript
- Marked.js for Markdown parsing
- Highlight.js for code syntax highlighting
- Material Icons for UI elements
